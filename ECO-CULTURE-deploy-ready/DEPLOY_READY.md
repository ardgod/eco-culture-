# ECO-CULTURE Deploy Rápido

## Deploy en Vercel

1. Sube este proyecto a GitHub
2. Ve a https://vercel.com
3. Importa el repositorio
4. Root Directory:
   frontend

## Build command
npm run build

## Install command
npm install --legacy-peer-deps

## Output directory
build
